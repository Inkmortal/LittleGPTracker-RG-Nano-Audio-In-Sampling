# RG Nano Installation Instructions

## Requirements
- Anbernic RG Nano (240x240 screen)
- Custom firmware (GarlicOS, muOS, or similar)
- SD card with enough space

## Installation

1. Copy `lgpt-rgnano.opk` to the SD card's `Native games` folder for the normal RG Nano launcher.

2. For loose ELF testing, copy `lgpt-rgnano.elf`, `config.xml`, `lgpt.funkey-s.desktop`, and `lgpt.png` to `/mnt/Applications/LGPT`.

3. Create folders for your projects and sample library if they do not already exist:
   `/mnt/Applications/Tracks`
   `/mnt/Applications/Samples`

4. Put `.wav` samples in `/mnt/Applications/Samples`.

5. Launch `LGPT RG Nano` from the RG Nano app menu.

## Controls

### Display Modes
- **L + FN/Select**: Toggle between Split-Row and Paged modes
  - **Split-Row** (default): See all 8 channels at once, 2 rows per song row
  - **Paged**: Show 4 channels at a time with auto-paging

### Paged Mode Navigation
- **Auto-paging**: View automatically follows cursor
  - Channels 0-3: Shows channels 1-4
  - Channels 4-7: Shows channels 5-8
- **L + Left**: Manually show channels 1-4
- **L + Right**: Manually show channels 5-8

### Standard Controls
See main LGPT documentation for full control reference.
Key mappings are optimized for the RG Nano button layout:
- Physical FN/Select is LGPT Select.
- Physical Menu/Power opens the LGPT app menu.
- **R + FN/Select** opens the LGPT helper/minimap from every LGPT screen.
- Hardware logs report FN/Select as `KEY_O` on this RG Nano OS path; the app config maps that to LGPT Select.

## Configuration

Edit `config.xml` to customize:
- `CHANNELMODE`: Set to "SPLIT" or "PAGED" for default mode
- `AUDIOBUFFERSIZE`: Adjust if audio glitches (512 is good default)
- `FONTTYPE`: Choose font (0=original, 1=digital, 2=monster)

## Troubleshooting

### Screen is cropped or wrong size
- Verify SCREENMULT is set to '1'
- Check firmware display settings

### Audio stuttering
- Increase AUDIOBUFFERSIZE (try 1024)
- Close other running applications

### Can't see all channels
- Press L + SELECT to toggle display mode
- In Paged mode, use L + Left/Right to switch banks

### Hardware session log
- LGPT writes diagnostics to `/mnt/Applications/lgpt-rgnano.log`
- The log records inputs, selected fields, screen state, playback state, and replay commands

## Performance Notes

RG Nano specifications:
- CPU: ARM Cortex-A7 @ 1.2GHz
- RAM: 64MB
- Screen: 240×240 (1.54" square IPS)

This is plenty for LGPT! Expect smooth performance with:
- All 8 channels active
- Complex patterns and tables
- Real-time effects and filters
- Sample playback

## Tips

1. **Split-Row Mode** is best for:
   - Composition (see all channels)
   - Live performance (mute visibility)
   - Complex arrangements

2. **Paged Mode** is best for:
   - Simple songs (4 channels or less)
   - When you want more visible rows
   - Traditional tracker feel

3. Use L + FN/Select to quickly switch modes based on what you're working on!
